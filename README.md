# fern-bookmarks.vim

[![fern plugin](https://img.shields.io/badge/ðŸŒ¿%20fern-plugin-yellowgreen)](https://github.com/lambdalisue/fern.vim)

[fern.vim](https://github.com/lambdalisue/fern.vim) plugin to add simple bookmarks for your favorite folders. Navigate to bookmark by opening a promt and typing the bookmark number. Bookmarks are stored `~/.FernBookmarks` by default.

![demo](bookmarks.gif)

## Usage

| Mapping | Action                      | Description                    |
| ------- | --------------------------- | ------------------------------ |
| `ms`    | `fern-action-bookmarks`     | Open bookmarks                 |
|         | `fern-action-bookmarks-add` | Add current node to bookmarks  |
